PWA vs Capacitor web wrapper
* PWA easy install 
* Capacitor in app store
* Cap has access to android APIs 


For embedding in Android:
Capacitor + (plain JS or React/Vue/Svelte) is a common, solid combination.

Recommendation
Stick with plain JS for now (it’s simplest and most robust).
If you want Android distribution next, do PWA first; if you need Play Store / APK, go Capacitor wrapper.
Move to a framework only when you’re adding settings, multiple layouts, personalization, persistence, and more UI complexity—then pick Svelte (lightweight) or React (ecosystem).
